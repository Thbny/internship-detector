# 🛡 Internship Detector

A SaaS-style web application that detects fraudulent internship postings using risk scoring, keyword analysis, blacklist verification, and audit logging.

## Features

- Scam internship detection logic
- Risk scoring system
- Suspicious keyword detection
- Blacklisted domain verification
- Government verification (CSV-based)
- Risk meter visualization
- User authentication (Login/Register)
- Session handling
- Audit logging

## Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas
- Git & GitHub

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
